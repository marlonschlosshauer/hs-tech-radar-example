Java Akka

View PDF

Anforderungen an moderne Systeme Wachsen stets gleichzeitig scheint der einzige Weg den Erwartungen gerecht zu werden, ein hoher Einsatz von Parallelität zu sein, welcher vor allem in großen Systemen eine immense Fehlerquelle darstellen kann. Akka ist ein Framework, welches es ermöglicht, Parallelität dank reaktiven Eigen- schaften zu vereinfachen und es Entwicklern erlaubt, sich auf das eigentliche Problem statt dem unterliegenden System zu konzentrieren. In diesem Paper wird Akka und dessen Komponen- ten vorgestellt und ein erster Eindruck über das Anwenden des Frameworks, für die Architektur einer modernen Anwendung gegeben.

React Native

Lange war es bei Smartphone Apps notwendig, sowohl für Android als auch IOS eine eigene App zu entwick- eln. Da Android Apps in Java/Kotlin jedoch IOS Apps in Objective-C/Swift geschrieben werden, war der Aufwand zum Veröffentlichen einer App auf beiden statt einem Betriebssystem praktisch doppelt so hoch. 2015 hat facebook mit React Native ein mobiles Framework veröf- fentlicht, welches auf dem erfolgreichen Web-Framework React basiert. React Native erlaubt es, mithilfe einer in Java-/Typescript geschriebenen Code-Basis eine sich nahezu identisch verhaltende Android und IOS App zu generieren. Dank Erweiterungen wie "react-native-web" und "react-native-windows" können aus React Native Apps ebenfalls Web-, Windows- und MacOS-App generiert werden. Im Laufe dieser Arbeit soll das Schreiben von React Native Apps anhand einer vollständigen Demo App erläutert werden.
